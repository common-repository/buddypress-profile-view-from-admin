This plugin allows admin user to view buddypress profile from admin amd will not work without buddypress.

== Description ==
This buddypress plugin allows admin  user to view a member profile page from admin user list page.

== Installation ==
1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the ‘Plugins’ SubPanel in WordPress
3. After plugin is activated it will create action link "View Profile" in users list page from where you can view users profile.
